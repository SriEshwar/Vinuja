Data Binding

Types of Data Binding

  • One-Way Binding
  • Two-Way Binding

One-Way Binding

Data flows in a single direction, either from the component to the view or from the view to the component.

Types of One-Way Binding

Note: The binding Given below ----- Component to view

  • Interpolation
  • Property
  • Class
  • Style
  • Attribute
  • Event

Interpolation Binding

Bind data from component to the view using {{expression } }.

Example for Interpolation Binding

Name: {{Name}}

College: {{CollegeName}}

Property Binding

Used to bind component properties to element properties, attributes, or DOM properties.

Example for Property Binding



Style Binding

Style binding is used to set individual CSS properties directly on an element. This is useful for applying inline styles dynamically.

Example for Style Binding

Welcome to learn Style Binding

Class Binding

Adds or removes one or more CSS classes from an element based on component properties.

Example for Class Binding

Text color

Attribute Binding

Adds or removes one or more CSS classes from an element based on component properties.

Example for Attribute Binding

Event Binding

Form of one-way binding because it flows data in one direction: from the template (view) to the component class (model).
